# Data Types

| Data Type | Example Values | Mutable | Ordered |
| --- | --- |---|---|
| str | 'Hello', "Hello" | No | Yes |
|int| 10, -10 | No | Yes |
|bool | True, False | No |
|list | [1,2,3] | Yes | Yes |
|dict | {"key":"value"} | Yes | No |


Dot method to access object attributes:
``` 
"john".upper()
>> JOHN
```
